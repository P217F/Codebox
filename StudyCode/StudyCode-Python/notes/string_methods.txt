ðŸ”¹ 1. String Property Check Functions
Function            Description
str.isalpha()   Returns True if all characters are letters
str.isdigit()   Returns True if all characters are digits
str.isalnum()   Returns True if all characters are alphanumeric
str.islower()   Returns True if all letters are lowercase
str.isupper()   Returns True if all letters are uppercase
str.isspace()   Returns True if all characters are whitespace
str.istitle()   Returns True if each word starts with an uppercase letter

ðŸ”¹ 2. Case Conversion
Function            Description
str.lower()         Converts string to lowercase
str.upper()             Converts string to uppercase
str.title()             Capitalizes the first letter of each word
str.capitalize()        Capitalizes the first letter of the string
str.swapcase()      Swaps case: uppercase to lowercase and vice versa

ðŸ”¹ 3. Searching and Replacing
Function                    Description
str.find(sub)           Finds the first index of sub, or -1 if not found
str.rfind(sub)          Finds the last index of sub
str.index(sub)          Same as find(), but raises an error if not found
str.replace(old, new)   Replaces all occurrences of old with new
str.count(sub)          Counts how many times sub appears

ðŸ”¹ Splitting and Joining Strings
Function                Description
str.split(sep)      Splits the string by the separator into a list
str.rsplit(sep)     Same as split() but from the right
str.splitlines()        Splits string at line breaks
'sep'.join(list)        Joins a list of strings into one string using sep

ðŸ”¹ Whitespace and Padding
Function                Description
str.strip()             Removes whitespace from both ends
str.lstrip()        Removes whitespace from the left
str.rstrip()        Removes whitespace from the right
str.zfill(width)        Pads the string on the left with zeros
str.center(width)       Centers the string with padding
str.ljust(width)        Left-aligns the string with padding
str.rjust(width)        Right-aligns the string with padding

ðŸ”¹ Start/End Checking
Function                    Description
str.startswith(prefix)  Returns True if string starts with prefix
str.endswith(suffix)    Returns True if string ends with suffix

ðŸ”¹ String Formatting
Method  Description
"{}".format(value)          Old-style formatting
f"{value}"                  F-string (modern and preferred way)
