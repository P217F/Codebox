- Lib Python need to study(5 -> 12):

os: provides a way to interact with the operating system.
subprocess: used to execute shell commands or external programs from within Python scripts
re: regular expression operations.
socket: low-level networking inteface
socketserver: a framework for network server
sys: access system-specific parameters and functions
time/datetime: time access and conversions/basic date and time types
argparse: command-line option and argument parsing library
scapy: powerful interactive packet manipulation library
pwntools: a CTF framework and exploit development library
threading: thread-based parallelism.
